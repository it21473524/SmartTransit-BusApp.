# Smart Transit – Bus Self-Care Application
Smart Transit – Bus Self-Care Application is a mobile app designed to enhance the urban bus travel experience in Sri Lanka. The application focuses on improving real-time bus tracking, ticket booking, accessibility, and travel planning for commuters.
## Project Overview
This project is developed as part of the **IT4031 VAUED Assignment 01 – 2025**.  
Our team was tasked by the **Sri Lanka Transport Board** to design an advanced mobile application for **booking and tracking bus journeys**. The application aims to improve **real-time tracking, ticket booking, accessibility**, and overall user experience for bus travelers.

The project focuses on:
- Searching for bus routes by origin, destination, and date
- Viewing real-time bus tracking with estimated arrival times
- Booking seats for long-distance or premium routes
- Multiple payment options (credit/debit card, mobile wallets)
- Receiving notifications on delays, cancellations, and diversions
- Downloading digital tickets (QR code or PDF)
- Planning multi-modal travel (bus + train)
- Providing accessibility features
